## Description:

A media streaming app to watch and listen to music, movies and tv shows from the 50s - 90s. Jump back to the past with Roku Flashback.

## Installation

Install node_modules with npm

```bash
  npm install  
```
Run the RokuUms

```bash
  node start  
```

Run the rokufrontend

```bash
  node start  
```
